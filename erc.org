We are using the emacs IRC client =ERC=.

We will load password sensitive information from a file not in the
repository

#+begin_src emacs-lisp
(load "~/.emacs.d/.erc-auth")
#+end_src

We want to connect to freenode upon entering =C-c e f=

#+begin_src emacs-lisp
(global-set-key "\C-cef" (lambda () (interactive)
			   (erc :server "irc.freenode.net" :port "6667"
				:nick irc-nick :password irc-password)))
#+end_src

We want to automatically join the following list of channels

#+begin_src emacs-lisp
(setq erc-autojoin-channels-alist '(("freenode.net" "#moenenspace" "#m-industries")))
#+end_src
